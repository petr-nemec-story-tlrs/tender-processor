# Insight Framework — From Data to Creative Concepts

## What Makes a Good Insight

An insight is NOT:
- A data point ("They have 477 followers")
- An observation ("Their engagement is low")
- A recommendation ("They should post more Reels")

An insight IS:
- A tension between what is and what could be
- A surprising truth that reframes the situation
- A human truth that connects data to behavior

**Formula:** `[Surprising data point] + [Why it matters for the audience] + [The opportunity it creates]`

## Insight Development Process

### Step 1: List all data points from research
Go through research files and list every notable finding:
- Performance metrics (engagement, followers, frequency)
- Content patterns (what works, what doesn't)
- Competitive gaps (what no one is doing)
- Audience signals (comments, sentiment, behavior)
- Market context (trends, cultural moments)

### Step 2: Find tensions and contradictions
Look for:
- **Say-do gaps:** Brand says X but content shows Y
- **Performance paradoxes:** Low effort content outperforms high effort
- **Market gaps:** Category leader does nothing in area X
- **Audience mismatches:** Target audience is on platform A but brand only invests in platform B
- **Cultural tensions:** Product positioning vs. how people actually use/discuss it

### Step 3: Select top 3 insights
Criteria for selection:
1. **Actionable** — We can build a creative concept around it
2. **Defensible** — Backed by specific data points
3. **Differentiating** — Not something every agency would say
4. **Scalable** — Works across multiple content formats and platforms

## From Insight to Creative Concept

Each insight gets ONE creative concept. A concept is:
- A **named idea** (2-4 words, memorable, can become a campaign name)
- A **one-sentence explanation** of what it means
- A **content approach** that brings it to life

**Example:**
- Insight: "Influencer posts get 100-300x more engagement, but the brand posts 90% own content"
- Concept: "Oddly Real Voices" — Let real people tell the brand story instead of the brand telling it themselves
- Content approach: Shift to 60% creator/UGC content, 30% brand, 10% community reposts

## Sample Post Structure

For each concept, create 3 sample posts. Each post includes:

```
### Post {N}: {Post title}

**Platform:** Instagram / TikTok / Facebook
**Format:** Static / Carousel / Reel / Story / TikTok video
**Copy:**
"{The actual post copy, including hashtags}"

**Creative description:**
Detailed description of the visual. What the viewer sees. Colors, composition, subjects, text overlays, mood.

**Video script (if applicable):**
- 0:00-0:03: Hook — what grabs attention
- 0:03-0:10: Setup — context
- 0:10-0:25: Main content
- 0:25-0:30: CTA / outro

**Why this works:**
1-2 sentences connecting back to the insight
```

## Influencer Strategy

Propose 3 influencer collaboration options across all insights:

### Option structure:
```
**Collaboration model:** (ambassador / campaign series / one-off / UGC creator)
**Influencer tier:** (nano 1-10K / micro 10-50K / mid 50-200K / macro 200K+)
**Number of influencers:** X
**Platform focus:** Instagram / TikTok / both
**Content deliverables:** X posts + X stories + X reels per influencer
**Recommended profiles:** (3-5 specific names with handles, follower counts, and why they fit)
**Estimated reach:** X
**Budget indication:** low / medium / high
```

### Profile recommendation criteria:
- Audience demographics match target audience
- Content style aligns with brand positioning
- Engagement rate above category average
- No conflicting brand partnerships
- Active in the relevant Czech market
- Authentic connection to the product category

## Presentation Integration

These insights and concepts flow into the PPTX presentation:
- **Chapter 1 (THE LANDSCAPE):** Sets up the data that leads to insights
- **Chapter 2 ({CONCEPT NAME}):** Presents insights → concepts → sample posts → influencer strategy

The concept name from the strongest insight typically becomes the Chapter 2 title.
